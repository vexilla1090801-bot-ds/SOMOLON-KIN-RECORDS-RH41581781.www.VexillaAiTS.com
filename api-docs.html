Authentication

To access the SKRMG repository, you must authenticate using your Organization API Key and the SMRMG/RH41581781 handshake protocol.

cURL Request
curl -X POST "https://api.vexilla.skrmg.com/v1/auth" \
  -H "Authorization: Bearer YOUR_API_KEY" \
  -d '{"repo_id": "RH41581781"}'

Verify Identity

POST /verify/asset

This endpoint checks the digital fingerprint of a media asset against the Vexilla ledger to ensure IP sovereignty.

Example JSON Response
{
  "status": "VERIFIED",
  "owner": "SKRMG_CONGLOMERATE",
  "talent": "DJ_M3",
  "nas4_active": true,
  "timestamp": "2026-03-13T00:12:00Z"
}

NAS-4 Integration

For REALM Pictures partners, the NAS-4 library allows you to inject neural-protection watermarks directly into your render frames.

Python Implementation
import vexilla_shield

# Initialize shield with RH41581781 cert
shield = vexilla_shield.Shield(cert="SMRMG_RH_PROT")
shield.inject_watermark(frame_data, mode="ultra_secure")
print("Frame secured by SKRMG")
